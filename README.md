# 🟩 Ranked Wordle w/ Daily Leaderboard!

A polished, full-stack Wordle clone with a daily leaderboard. Solve the daily mystery word, race against the clock, and climb the leaderboard.

---

## ✨ Features

- **Daily Word** - One word per day, synced for all players worldwide.
- **Classic Gameplay** - 6 attempts with color-coded feedback (Green, Orange, Grey).
- **Interactive UI** - Animated tile flips, "pop" effects on keystrokes, and shake alerts for invalid words.
- **Live Leaderboard** - Competitive ranking based on **fewest guesses** first, then **fastest time**.
- **Virtual Keyboard** - Dynamic key coloring based on your best results.
- **Live Timer** - Precision tracking of your solve speed.

---

## 🛠 Tech Stack

- **Frontend**: React 18 & Axios
- **Backend**: Node.js & Express
- **Database**: MongoDB (via Mongoose)
